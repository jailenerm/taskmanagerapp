# taskmanagerapp
Assignment task management app built with React Native and Expo
A mobile assignment and task management app built with React Native and Expo.

## Features
- Color-coded courses
- Push notifications for due dates
- Progress tracker per course
- Study timer (Pomodoro)
- Test and exam reminders

## How to run
1. Clone this repo
2. Run `npm install`
3. Run `npx expo start`
4. Press `i` for iOS Simulator

## Built with
- React Native + Expo
- JavaScript
- AsyncStorage
- Expo Notifications
